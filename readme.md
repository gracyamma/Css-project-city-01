# 🚀 Landing Page – DAO Token Project

A modern, responsive landing page built using **HTML** and **Tailwind CSS**, designed to promote a DAO-based token ecosystem. This project focuses on clean UI, typography, and responsive design.

---

[live@](./https://gracyamma.github.io/Css-project-city-01/)




![image](./chrome-capture-2026-04-24%20(1).png)

## 📌 Features

* ⚡ Fully responsive layout (mobile → desktop)
* 🎨 Styled with Tailwind CSS
* 🔤 Custom fonts using Google Fonts (Kanit, Poppins, Momo Signature)
* 🧭 Navigation bar with menu items and CTA
* 💡 Hero section with strong messaging
* 🔘 Call-to-action buttons
* 🎯 Minimal and modern UI design

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS (CDN)
* Font Awesome Icons
* Google Fonts

## ⚙️ Setup & Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the project folder:

   ```bash
   cd your-repo-name
   ```

3. Open `index.html` in your browser.

---

## 🖼️ Customization

* Replace the background image:

  ```css
  background-image: url(./your-image.jpeg);
  ```

* Update text content inside `<main>` section.

* Modify colors using Tailwind classes:

  * Example: `bg-pink-600` → `bg-blue-600`

---

## 📱 Responsive Design

* Uses Tailwind breakpoints:

  * `lg:` for large screens
  * `xl:` for extra large screens
